## Binary Search

* 정렬된 일련의 값들이 주어졌을 때 그들 중에서 원하는 값을 찾는 알고리즘
* 시간복잡도 *O(logN)*

## Parametric Search

* 이진 탐색과 비슷하다.
* 주어진 값이 아닌 범위를 기준으로 답을 탐색한다.
* ex) M미터 이상의 나무를 얻기 위한 절단기의 최댓값(2805.cpp),<br>
 N개를 만들 수 있는 랜선의 최대길이(1654.cpp)
* 따라서 상황에 맞게 범위를 고려해서 코드를 구현하면 된다.
